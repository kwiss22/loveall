# CSS가 로드되지 않는 문제 해결

## 🔍 문제 진단

브라우저에서 "검은 글씨"만 보인다면 CSS 파일이 제대로 로드되지 않은 것입니다.

---

## ✅ 해결 방법

### 방법 1: 브라우저 새로고침 (가장 간단)

1. **브라우저에서 Ctrl + Shift + R** (하드 리프레시)
   - 또는 **Ctrl + F5**
   - 또는 **F12** 키를 누르고 새로고침 버튼을 오른쪽 클릭 → "캐시 비우기 및 강력 새로고침"

2. **다시 접속:**
   ```
   http://127.0.0.1:5000
   ```

### 방법 2: CSS 파일 직접 확인

브라우저에서 CSS 파일이 제대로 로드되는지 확인:

1. **브라우저 주소창에 입력:**
   ```
   http://127.0.0.1:5000/static/style.css
   ```

2. **CSS 내용이 보이면 정상:**
   - CSS 코드가 그대로 보이면 정상입니다
   - "Not Found" 오류가 나면 문제가 있습니다

### 방법 3: 브라우저 개발자 도구 확인

1. **F12 키 누르기** (개발자 도구 열기)

2. **"Console" 탭 확인:**
   - 빨간색 오류 메시지가 있는지 확인
   - CSS 파일 로드 오류가 있는지 확인

3. **"Network" 탭 확인:**
   - 새로고침 (F5)
   - `style.css` 파일이 로드되는지 확인
   - 200 OK 상태인지 확인

## 🔧 추가 확인 사항

### 1. Flask 앱이 올바르게 실행 중인가?

터미널에서 다음 메시지가 보여야 합니다:
```
* Running on http://127.0.0.1:5000
* Debug mode: on
```

**중요:** `web_vocab_app.py`가 실행 중이어야 합니다!
- ❌ `test_flask_simple.py` (X) - 이것은 테스트용
- ✅ `web_vocab_app.py` (O) - 이것이 실제 앱

### 2. 올바른 주소로 접속했나?

- ✅ `http://127.0.0.1:5000` (O)
- ❌ `http://127.0.0.1:5000/main` (X) - 이 경로는 없습니다

### 3. 파일 구조 확인

다음 파일들이 있어야 합니다:
```
game_english/
├── web_vocab_app.py
├── templates/
│   └── index.html
└── static/
    ├── style.css  ← 이 파일이 있어야 합니다!
    └── script.js
```

## 💡 단계별 해결

### 1단계: Flask 앱 재시작

1. **터미널에서 Ctrl + C**로 Flask 앱 중지

2. **다시 실행:**
   ```powershell
   python web_vocab_app.py
   ```

3. **성공 메시지 확인:**
   ```
   * Running on http://127.0.0.1:5000
   ```

### 2단계: 브라우저 캐시 지우기

1. **Ctrl + Shift + Delete** 누르기
2. "캐시된 이미지 및 파일" 선택
3. "데이터 삭제" 클릭
4. 브라우저 닫기

### 3단계: 브라우저 다시 열기

1. **새 브라우저 창 열기**
2. **주소 입력:**
   ```
   http://127.0.0.1:5000
   ```
3. **Enter 키 누르기**

### 4단계: 개발자 도구로 확인

1. **F12 키** 누르기
2. **Console 탭** 확인
3. **빨간색 오류가 있으면** 내용 확인

## 🔍 CSS가 로드되지 않는 이유

1. **브라우저 캐시 문제**
   - 해결: 하드 리프레시 (Ctrl + Shift + R)

2. **Flask 앱이 올바르게 실행되지 않음**
   - 해결: `web_vocab_app.py`를 올바르게 실행

3. **잘못된 주소로 접속**
   - 해결: `http://127.0.0.1:5000`로 접속

4. **파일 구조 문제**
   - 해결: `static/style.css` 파일이 있는지 확인

## ✅ 성공 확인

CSS가 제대로 로드되면:
- **보라색 그라데이션 배경**이 보여야 합니다
- **흰색 카드 형태의 컨테이너**가 보여야 합니다
- **보라색 헤더**가 보여야 합니다
- **탭 버튼**이 보여야 합니다
- **검은 글씨만 보이는 게 아니라** 예쁘게 디자인된 화면이 보여야 합니다
