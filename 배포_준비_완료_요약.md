## 📖 사용 방법

### 로컬에서 실행
```bash
pip install -r requirements.txt
python web_vocab_app.py
```

### 프로덕션 배포
```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 web_vocab_app:app
```

자세한 내용은 `DEPLOYMENT.md` 파일을 참고하세요.
