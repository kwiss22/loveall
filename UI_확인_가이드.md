# UI 확인 가이드

## ❌ 현재 보이는 화면 (틀림)

브라우저에 보이는 화면:
- "Flask 테스트 성공!" 메시지
- "웹 서버가 정상적으로 작동하고 있습니다."
- "이제 메인 앱으로 이동하거나"
- "web_vocab_app.py를 실행해보세요."

**이것은 실제 단어장 UI가 아닙니다!**
- 이것은 `test_flask_simple.py`의 화면입니다
- 실제 단어장 기능이 없습니다
- 단순한 테스트 메시지만 표시됩니다

---

## ✅ 올바른 UI (맞음)

실제 단어장 UI는 다음과 같아야 합니다:

### 화면 구성:
1. **보라색 그라데이션 배경**
   - 화면 전체가 보라색에서 보라색으로 그라데이션

2. **흰색 카드 형태의 컨테이너**
   - 중앙에 둥근 모서리의 흰색 박스

3. **보라색 헤더**
   - 상단에 보라색 그라데이션 헤더
   - "📖 영어 단어장" 제목
   - "총 X개의 단어가 저장되어 있습니다." 부제목

4. **탭 버튼 (4개)**
   - "단어 목록" (활성화됨)
   - "단어 추가"
   - "퀴즈"
   - "통계"
   - 회색 배경에 보라색 강조

5. **단어 목록**
   - 저장된 단어들이 카드 형태로 표시
   - 각 단어 카드:
     - 영어 단어 (큰 글씨, 굵게)
     - 한국어 뜻 (작은 글씨)
     - 삭제 버튼 (빨간색)

---

## 🔧 해결 방법

### 1단계: 브라우저 완전히 새로고침

1. **Ctrl + Shift + R** (하드 리프레시)
   - 또는 **Ctrl + F5**
   - 또는 **F12** → 새로고침 버튼 오른쪽 클릭 → "캐시 비우기 및 강력 새로고침"

2. **주소 확인:**
   ```
   http://127.0.0.1:5000
   ```
   - 슬래시(`/`) 뒤에 아무것도 없어야 합니다

### 2단계: 브라우저 캐시 완전히 지우기

1. **Ctrl + Shift + Delete** 누르기
2. **캐시 삭제** 선택
3. **브라우저 완전히 닫기**
4. **새 브라우저 창 열기**
5. **주소 입력:** `http://127.0.0.1:5000`
6. **Enter 키**

### 3단계: 다른 Flask 앱이 실행 중인지 확인

1. **모든 터미널 창 확인**
   - VS Code 터미널
   - PowerShell 창
   - 다른 터미널 창들

2. **실행 중인 Flask 앱 찾기**
   - "Running on http://127.0.0.1:5000" 메시지가 있는 모든 창 확인

3. **`test_flask_simple.py`가 실행 중이면:**
   - 해당 터미널에서 **Ctrl + C**로 중지

4. **`web_vocab_app.py`만 실행 중인지 확인:**
   - 터미널에 "영어 단어장 웹 애플리케이션 시작!" 메시지가 보여야 합니다

### 4단계: 시크릿 모드로 확인

1. **Ctrl + Shift + N** (Chrome) 또는 **Ctrl + Shift + P** (Edge)
2. **주소 입력:** `http://127.0.0.1:5000`
3. **Enter 키**
4. 올바른 UI가 보이는지 확인

## ✅ 정상 UI 확인 체크리스트

올바른 UI가 보이면:

- [ ] **보라색 그라데이션 배경**이 보이나요?
- [ ] **흰색 카드 형태의 컨테이너**가 중앙에 있나요?
- [ ] **"영어 단어장" 제목**이 보라색 헤더에 있나요?
- [ ] **탭 버튼 4개** (단어 목록, 단어 추가, 퀴즈, 통계)가 보이나요?
- [ ] **단어 목록**이 카드 형태로 표시되나요?
- [ ] **예쁘게 디자인된 화면**이 보이나요?

## 🔍 문제가 계속되면

### 확인 사항:

1. **터미널 확인:**
   - "영어 단어장 웹 애플리케이션 시작!" 메시지가 보이나요?
   - "web_vocab_app"이 실행 중인가요?

2. **브라우저 주소 확인:**
   - 정확히 `http://127.0.0.1:5000`인가요?
   - `/main` 같은 경로가 붙어있지 않은가요?

3. **브라우저 개발자 도구 (F12) 확인:**
   - Console 탭: 오류가 있나요?
   - Network 탭: `style.css`가 로드되나요? (200 OK?)

4. **다른 Flask 앱이 실행 중인가요?**
   - `test_flask_simple.py`가 실행 중이면 Ctrl + C로 중지

## 💡 가장 확실한 방법

1. **모든 터미널에서 Flask 앱 중지:**
   - 모든 창에서 Ctrl + C

2. **브라우저 완전히 닫기**

3. **Flask 앱 다시 실행:**
   ```powershell
   python web_vocab_app.py
   ```

4. **성공 메시지 확인:**
   - "영어 단어장 웹 애플리케이션 시작!" 확인

5. **새 브라우저 창 열기**

6. **주소 입력:**
   ```
   http://127.0.0.1:5000
   ```

7. **Enter 키**

8. **보라색 그라데이션 배경과 예쁜 UI 확인!**
