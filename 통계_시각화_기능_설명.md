# 통계 시각화 기능 설명

## 📊 추가된 기능

### 1. 정답률 상위 10개 단어 막대 그래프
- **차트 타입**: 막대 그래프 (Bar Chart)
- **표시 내용**: 정답률이 높은 상위 10개 단어
- **기능**:
  - 각 단어의 정답률을 시각적으로 비교
  - 마우스 호버 시 상세 정보 표시 (정답률, 맞춘 횟수, 틀린 횟수)
  - 단어 이름이 길면 자동으로 줄임

### 2. 전체 통계 요약 도넛 차트
- **차트 타입**: 도넛 차트 (Doughnut Chart)
- **표시 내용**: 전체 정답/오답 비율
- **기능**:
  - 전체 퀴즈에서 정답과 오답의 비율을 한눈에 파악
  - 마우스 호버 시 상세 정보 표시 (횟수, 비율)
  - 색상으로 구분 (초록색: 정답, 빨간색: 오답)

## 🎨 디자인 특징

### 다크 모드 지원
- 라이트 모드와 다크 모드 모두 지원
- 모드 전환 시 차트 색상 자동 업데이트
- 차트 텍스트와 그리드 색상도 모드에 맞게 조정

### 반응형 디자인
- 데스크톱: 2개 차트를 나란히 표시
- 모바일: 차트를 세로로 배치
- 화면 크기에 따라 자동 조정

## 💻 기술 스택

### Chart.js 라이브러리
- **버전**: 4.4.0
- **CDN**: jsdelivr.net을 통해 로드
- **특징**: 
  - 가볍고 빠름
  - 다양한 차트 타입 지원
  - 커스터마이징 가능

## 🔧 구현 세부사항

### 차트 생성 로직
1. 통계 데이터를 API에서 가져옴
2. 데이터를 정렬하고 필터링
3. Chart.js로 차트 생성
4. 다크 모드에 맞게 색상 설정

### 차트 업데이트
- 통계 탭을 열 때마다 최신 데이터로 차트 생성
- 다크 모드 전환 시 차트 재생성
- 기존 차트가 있으면 파괴 후 새로 생성 (메모리 누수 방지)

## 📈 사용 방법

1. **통계 탭 클릭**
   - 상단 네비게이션에서 "통계" 탭 선택

2. **차트 확인**
   - 정답률 상위 10개 단어 막대 그래프 확인
   - 전체 통계 요약 도넛 차트 확인

3. **상세 정보 보기**
   - 차트 위에 마우스를 올리면 상세 정보 표시
   - 막대 그래프: 단어별 정답률과 정/오답 횟수
   - 도넛 차트: 전체 정답/오답 비율

4. **다크 모드 전환**
   - 헤더의 다크 모드 버튼 클릭
   - 차트 색상이 자동으로 업데이트됨

## 🎯 학습 효과

### 시각적 피드백
- 숫자만 보는 것보다 그래프로 보면 이해가 쉬움
- 어떤 단어를 더 공부해야 하는지 한눈에 파악

### 학습 동기 부여
- 정답률이 높은 단어를 보며 성취감 느끼기
- 부족한 부분을 시각적으로 확인하여 집중 학습

## 🔮 향후 개선 가능한 기능

1. **더 많은 차트 타입**
   - 시간별 정답률 추이 (라인 차트)
   - 단어별 학습 횟수 (히스토그램)

2. **인터랙티브 기능**
   - 차트에서 단어 클릭 시 상세 정보 표시
   - 차트 확대/축소 기능

3. **필터링 기능**
   - 기간별 통계 보기
   - 특정 단어만 선택하여 차트 표시

4. **데이터 내보내기**
   - 차트를 이미지로 저장
   - 통계 데이터를 CSV로 내보내기
