# 실행이 안 될 때 확인 사항

## 🔍 어떤 오류가 발생하나요?

실행이 안 되는 이유는 여러 가지가 있을 수 있습니다. 
터미널에 나오는 **정확한 오류 메시지**를 알려주시면 더 정확하게 도와드릴 수 있습니다.

---

## 📋 일반적인 문제와 해결 방법

### 문제 1: 명령어를 잘못 입력함

**증상:**
```
'web_vocab_app.py' is not recognized as the name of a cmdlet...
```

**원인:** `python` 없이 파일 이름만 입력

**해결:**
```powershell
# ❌ 잘못된 방법
web_vocab_app.py

# ✅ 올바른 방법
python web_vocab_app.py
```

---

### 문제 2: Python을 찾을 수 없음

**증상:**
```
'python' is not recognized as the name of a cmdlet...
```

**원인:** Python이 설치되지 않았거나 PATH에 없음

**해결:**
1. Python 설치 확인:
   ```powershell
   python --version
   ```

2. 버전이 나오지 않으면 Python 설치 필요

---

### 문제 3: Flask가 설치되지 않음

**증상:**
```
ModuleNotFoundError: No module named 'flask'
```

**해결:**
```powershell
pip install flask
```

---

### 문제 4: 포트가 이미 사용 중

**증상:**
```
Address already in use
또는
OSError: [WinError 10048] 보통 각 소켓 주소는 한 번만 사용할 수 있습니다
```

**원인:** 다른 프로그램이 5000번 포트 사용 중

**해결:**
1. 다른 터미널에서 실행 중인 Flask 앱 확인
2. Ctrl + C로 종료
3. 다시 실행

또는 포트 변경:
- `web_vocab_app.py` 파일의 218번 줄을 수정:
  ```python
  app.run(debug=True, host='127.0.0.1', port=5001, use_reloader=False)
  ```

---

### 문제 5: 파일을 찾을 수 없음

**증상:**
```
FileNotFoundError: [Errno 2] No such file or directory: 'web_vocab_app.py'
```

**원인:** 잘못된 폴더에 있음

**해결:**
1. 올바른 폴더로 이동:
   ```powershell
   cd C:\Users\Admin\projects\game_english
   ```

2. 파일이 있는지 확인:
   ```powershell
   dir web_vocab_app.py
   ```

---

### 문제 6: 문법 오류

**증상:**
```
SyntaxError: invalid syntax
```

**원인:** 코드에 문법 오류가 있음

**해결:**
- 터미널에 나오는 오류 메시지를 확인
- 어떤 줄에서 오류가 발생했는지 확인
- 오류 내용을 알려주시면 수정 도와드립니다

---

## 🔧 진단 도구 사용하기

실행 진단 도구를 실행해보세요:

```powershell
python 실행_진단.py
```

이 스크립트가 문제를 자동으로 찾아줍니다.

## 💡 확인 순서

### 1단계: 명령어 확인
```powershell
python web_vocab_app.py
```
- `python`을 앞에 붙였는지 확인

### 2단계: 폴더 확인
```powershell
cd C:\Users\Admin\projects\game_english
```
- 올바른 폴더에 있는지 확인

### 3단계: 파일 확인
```powershell
dir web_vocab_app.py
```
- 파일이 있는지 확인

### 4단계: 실행
```powershell
python web_vocab_app.py
```
- 실행하고 오류 메시지 확인

## 📞 도움이 필요할 때

다음 정보를 알려주시면 더 정확하게 도와드릴 수 있습니다:

1. **어떤 명령어를 입력했나요?**
   - 예: `python web_vocab_app.py`

2. **터미널에 나온 오류 메시지는 무엇인가요?**
   - 빨간색 오류 메시지를 복사해서 알려주세요

3. **어떤 폴더에 있나요?**
   - 터미널에 나오는 경로 확인

4. **Python이 설치되어 있나요?**
   - `python --version` 결과 확인
