# 웹 앱 접속 가이드

## ✅ 올바른 접속 방법

### 1. Flask 앱이 실행 중인지 확인
터미널에 다음 메시지가 보여야 합니다:
```
* Running on http://127.0.0.1:5000
* Debug mode: on
```

### 2. 브라우저 주소창에 **정확히** 입력:
```
http://127.0.0.1:5000
```

**중요:**
- ✅ `http://127.0.0.1:5000` (O)
- ❌ `http://127.0.0.1:5000/main` (X) - 이 경로는 없습니다!
- ❌ `https://127.0.0.1:5000` (X) - https가 아닌 http 사용!

### 3. Enter 키 누르기

---

## 🔍 문제 해결

### 문제 1: "사이트에 연결할 수 없음" 오류

**가능한 원인:**
1. Flask 앱이 실행되지 않음
2. 잘못된 주소 입력
3. 포트가 다른 프로그램에 사용 중

**해결 방법:**

#### 1단계: Flask 앱이 실행 중인지 확인
터미널/VS Code 터미널에서:
```
* Running on http://127.0.0.1:5000
```
이 메시지가 보여야 합니다.

#### 2단계: 올바른 주소로 접속
브라우저 주소창에:
```
http://127.0.0.1:5000
```
**주의:** `/main` 같은 경로를 추가하지 마세요!

#### 3단계: 브라우저 새로고침
- F5 키 누르기
- 또는 Ctrl + Shift + R (하드 리프레시)

---

### 문제 2: "404 Not Found" 오류

**원인:** 존재하지 않는 경로로 접속

**해결:**
- 루트 경로(`/`)로만 접속하세요: `http://127.0.0.1:5000`
- 다른 경로(`/main`, `/test` 등)는 사용하지 마세요

---

### 문제 3: Flask 앱이 실행되지 않음

**해결 방법:**

1. **터미널에서 Flask 앱 실행:**
   ```powershell
   cd C:\Users\Admin\projects\game_english
   python web_vocab_app.py
   ```

2. **실행 확인:**
   다음 메시지가 보여야 합니다:
   ```
   ============================================================
   영어 단어장 웹 애플리케이션 시작!
   ============================================================
   * Running on http://127.0.0.1:5000
   ```

3. **중요:** 이 메시지가 보이는 동안 터미널 창을 닫지 마세요!

## 📋 체크리스트

접속하기 전에 확인:

- [ ] Flask 앱이 실행 중인가요?
  - 터미널에 "Running on http://127.0.0.1:5000" 메시지 확인
  
- [ ] 올바른 주소로 접속했나요?
  - `http://127.0.0.1:5000` (O)
  - `http://127.0.0.1:5000/main` (X)
  - `https://127.0.0.1:5000` (X)

- [ ] 터미널 창이 열려 있나요?
  - Flask 앱이 실행 중이어야 합니다
  
- [ ] 브라우저를 새로고침했나요?
  - F5 키 또는 Ctrl + R

## 🎯 단계별 접속 가이드

### 1단계: Flask 앱 실행 확인
VS Code 터미널 또는 PowerShell에서:
```powershell
python web_vocab_app.py
```

### 2단계: 성공 메시지 확인
다음 메시지가 보이면 성공:
```
* Running on http://127.0.0.1:5000
Press CTRL+C to quit
```

### 3단계: 브라우저 열기
1. Chrome, Edge, Firefox 등 브라우저 열기
2. 주소창 클릭
3. **정확히** 입력: `http://127.0.0.1:5000`
4. Enter 키 누르기

### 4단계: 웹 페이지 확인
다음과 같은 페이지가 보이면 성공:
- "영어 단어장" 제목
- "단어 목록", "단어 추가", "퀴즈", "통계" 탭
- 단어 목록이 표시됨

2. **다른 브라우저 시도:**
   - Chrome, Edge, Firefox 등 여러 브라우저로 시도

3. **터미널 메시지 확인:**
   - Flask 앱 실행 시 오류 메시지가 있는지 확인
   - 빨간색 오류 메시지가 있으면 내용을 확인

4. **포트 확인:**
   - 5000번 포트가 다른 프로그램에 사용 중일 수 있습니다
   - `web_vocab_app.py`에서 포트를 5001로 변경 가능

## 🔧 빠른 문제 해결

### 방법 1: 완전히 새로 시작
1. 터미널에서 Ctrl + C로 Flask 앱 종료
2. 다시 `python web_vocab_app.py` 실행
3. 브라우저 주소창에 `http://127.0.0.1:5000` 입력
4. Enter 키 누르기

### 방법 2: 브라우저 캐시 지우기
1. Ctrl + Shift + Delete
2. 캐시 지우기 선택
3. 다시 접속 시도

### 방법 3: 시크릿 모드로 접속
1. Ctrl + Shift + N (Chrome) 또는 Ctrl + Shift + P (Edge)
2. 주소창에 `http://127.0.0.1:5000` 입력
3. Enter 키 누르기

---
