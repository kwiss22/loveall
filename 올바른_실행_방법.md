# 올바른 Flask 앱 실행 방법

## 🔍 현재 상황

브라우저에 "Flask 테스트 성공!" 메시지가 보이신다면, **테스트용 파일**(`test_flask_simple.py`)이 실행 중입니다.

실제 **단어장 웹 앱**(`web_vocab_app.py`)을 실행해야 합니다!

---

## ✅ 올바른 실행 방법

### 1단계: 현재 실행 중인 Flask 앱 중지

터미널 창에서:
- **Ctrl + C** 키를 누르세요
- Flask 앱이 종료됩니다

### 2단계: 올바른 파일 실행

터미널에서 다음 명령어 실행:
```powershell
python web_vocab_app.py
```

**중요:**
- ❌ `test_flask_simple.py` (X) - 이것은 테스트용입니다
- ✅ `web_vocab_app.py` (O) - 이것이 실제 단어장 앱입니다!

### 3단계: 성공 메시지 확인

다음 메시지가 보이면 성공:
```
============================================================
영어 단어장 웹 애플리케이션 시작!
============================================================
단어 개수: 6개
통계 기록: 2개
접속 주소: http://127.0.0.1:5000
============================================================

* Running on http://127.0.0.1:5000
* Debug mode: on
```

### 4단계: 브라우저 접속

브라우저 주소창에 입력:
```
http://127.0.0.1:5000
```

**이번에는 다음과 같은 화면이 보여야 합니다:**
- "영어 단어장" 제목
- "단어 목록", "단어 추가", "퀴즈", "통계" 탭
- 저장된 단어 목록

## 📋 차이점 비교

### ❌ test_flask_simple.py (테스트용)
- 간단한 테스트 메시지만 표시
- "Flask 테스트 성공!" 메시지
- 실제 기능 없음

### ✅ web_vocab_app.py (실제 앱)
- 완전한 단어장 웹 앱
- 단어 추가/삭제/검색 기능
- 퀴즈 기능
- 통계 기능
- 파일 저장/불러오기

## 🔧 문제 해결

### 문제: "Flask 테스트 성공!" 메시지만 보임

**원인:** `test_flask_simple.py`가 실행 중

**해결:**
1. 터미널에서 Ctrl + C로 중지
2. `python web_vocab_app.py` 실행
3. 브라우저 새로고침

### 문제: 포트가 이미 사용 중

**원인:** `test_flask_simple.py`가 아직 실행 중

**해결:**
1. 모든 터미널 창 확인
2. Ctrl + C로 모든 Flask 앱 중지
3. 다시 `python web_vocab_app.py` 실행

### 문제: 어떤 파일이 실행 중인지 모름

**확인 방법:**
- 터미널 창을 확인하세요
- `test_flask_simple.py`가 보이면 → Ctrl + C로 중지
- `web_vocab_app.py`가 보여야 합니다!
