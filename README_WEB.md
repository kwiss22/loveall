# 영어 단어장 웹 버전 사용 가이드

## 📦 설치 및 실행

### 1. Flask 확인
Flask가 설치되어 있는지 확인:
```bash
python -c "import flask; print('Flask 설치됨')"
```

Flask가 없다면 설치:
```bash
pip install flask
```

### 2. 웹 앱 실행
```bash
python web_vocab_app.py
```

### 3. 브라우저에서 접속
브라우저를 열고 다음 주소로 접속:
```
http://127.0.0.1:5000
```

## 🎯 기능

### 1. 단어 목록
- 저장된 모든 단어를 카드 형태로 표시
- 각 단어에 삭제 버튼 제공
- 실시간으로 단어 개수 표시

### 2. 단어 추가
- 영어 단어와 한국어 뜻 입력
- 추가 후 자동으로 목록 갱신
- 성공/실패 메시지 표시

### 3. 퀴즈
- **영어 → 한글**: 영어 단어를 보고 한국어 뜻 맞추기
- **한글 → 영어**: 한국어 뜻을 보고 영어 단어 맞추기
- 정답/오답 즉시 확인
- Enter 키로 정답 확인 가능
- 다음 문제 버튼으로 연속 풀기

### 4. 통계
- 각 단어별 정답률 표시
- 맞춘 횟수 / 틀린 횟수 표시
- 정답률 순으로 정렬

## 🎨 주요 특징

- **반응형 디자인**: 모바일과 데스크톱 모두 지원
- **실시간 업데이트**: API를 통한 데이터 동기화
- **자동 저장**: 단어 추가/삭제 시 자동으로 파일에 저장
- **통계 추적**: 퀴즈 결과를 자동으로 기록하고 분석

## 💡 사용 팁

1. **단어 추가**: 영어 단어는 자동으로 소문자로 변환됩니다
2. **퀴즈**: Enter 키를 누르면 빠르게 정답을 확인할 수 있습니다
3. **통계 확인**: 퀴즈를 많이 풀수록 통계가 더 정확해집니다
4. **자동 저장**: 모든 변경사항은 즉시 파일에 저장됩니다

## 🔧 문제 해결

### 포트가 이미 사용 중일 때
에러 메시지: `Address already in use`

**해결 방법:**
1. 다른 프로그램이 5000번 포트를 사용하고 있는지 확인
2. `web_vocab_app.py` 파일의 마지막 줄에서 포트 번호 변경:
   ```python
   app.run(debug=True, host='0.0.0.0', port=5001)  # 5001로 변경
   ```

### 데이터가 보이지 않을 때
- `vocabulary.json` 파일이 같은 폴더에 있는지 확인
- 브라우저 새로고침 (F5)
- "새로고침" 버튼 클릭

### CSS/JS가 로드되지 않을 때
- `static` 폴더가 올바른 위치에 있는지 확인
- 브라우저 개발자 도구(F12)에서 콘솔 에러 확인
