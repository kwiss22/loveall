# 영어 단어장 프로그램 테스트 가이드

## 1단계: 프로그램 실행

PowerShell이나 명령 프롬프트를 열고 다음 명령어를 입력하세요:

```bash
cd C:\Users\Admin\projects\game_english
python vocab_book.py
```

## 2단계: 단계별 테스트

### 테스트 1: 단어 추가하기
1. 화면에 메뉴가 표시되면 `1` 입력 후 Enter
2. "영어 단어를 입력하세요:" → `apple` 입력 후 Enter
3. "한국어 뜻을 입력하세요:" → `사과` 입력 후 Enter
4. "[OK] 'apple' 단어가 추가되었습니다!" 메시지 확인

**추가로 더 입력해보세요:**
- `book` → `책`
- `cat` → `고양이`
- `dog` → `강아지`

### 테스트 2: 단어 목록 보기
1. 메뉴에서 `2` 입력 후 Enter
2. 추가한 모든 단어가 목록에 표시되는지 확인
3. "총 X개의 단어가 저장되어 있습니다." 메시지 확인

### 테스트 3: 단어 검색하기
1. 메뉴에서 `3` 입력 후 Enter
2. "검색할 단어를 입력하세요:" → `apple` 입력 후 Enter
3. "[OK] apple → 사과" 메시지 확인

**존재하지 않는 단어 검색해보기:**
- `zebra` 입력 → "[ERROR] 'zebra' 단어를 찾을 수 없습니다." 확인

### 테스트 4: 퀴즈 - 영어 → 한글
1. 메뉴에서 `5` 입력 후 Enter
2. 영어 단어가 표시됩니다 (예: "영어 단어: apple")
3. "한국어 뜻을 입력하세요:" → `사과` 입력 후 Enter
4. 정답 시 "[OK] 정답입니다!" 메시지 확인
5. 오답 시 "[WRONG] 틀렸습니다." 메시지 확인

**여러 번 시도해보세요:**
- 맞추기도 해보고
- 일부러 틀려보기도 해보세요

### 테스트 5: 퀴즈 - 한글 → 영어
1. 메뉴에서 `6` 입력 후 Enter
2. 한국어 뜻이 표시됩니다 (예: "한국어 뜻: 책")
3. "영어 단어를 입력하세요:" → `book` 입력 후 Enter
4. 정답/오답 메시지 확인

### 테스트 6: 퀴즈 통계 보기
1. 메뉴에서 `7` 입력 후 Enter
2. 각 단어별로 정답 횟수/오답 횟수/정답률이 표시되는지 확인
3. 퀴즈를 더 풀어보고 다시 통계를 확인해보세요 (정답률이 변경되는지 확인)

### 테스트 7: 파일 저장하기
1. 메뉴에서 `8` 입력 후 Enter
2. "[OK] 파일 저장 완료!" 메시지 확인
3. 파일 탐색기에서 `vocabulary.json` 파일이 생성되었는지 확인
4. 파일을 열어서 내용이 JSON 형식으로 저장되었는지 확인

### 테스트 8: 프로그램 종료 후 재실행
1. 메뉴에서 `0` 입력 후 Enter
2. "종료 전에 파일에 저장하시겠습니까? (y/n):" → `y` 입력 후 Enter
3. 프로그램이 종료됩니다
4. **다시 프로그램 실행**: `python vocab_book.py`
5. 프로그램 시작 시 자동으로 파일이 불러와지는지 확인
6. 메뉴 2로 단어 목록을 확인해서 이전에 추가한 단어들이 그대로 있는지 확인

## 3단계: 예상 결과 확인

### 정상 작동 시:
- ✅ 모든 메뉴가 정상적으로 작동
- ✅ 단어 추가/삭제/검색 정상
- ✅ 퀴즈가 랜덤으로 단어 선택
- ✅ 통계가 정확하게 기록됨
- ✅ 파일 저장/불러오기 정상

### 문제 발생 시:
- 프로그램이 오류 메시지와 함께 종료되는지 확인
- 오류 메시지 내용을 기록해두세요
- 어떤 메뉴에서 문제가 발생했는지 확인

## 4단계: 추가 테스트 아이디어

1. **빈 입력 테스트**: 단어 추가 시 아무것도 입력하지 않고 Enter
2. **대소문자 테스트**: 영어 단어를 대문자로 입력해보기 (자동으로 소문자로 변환되는지 확인)
3. **공백 처리**: 단어 입력 시 앞뒤 공백이 자동으로 제거되는지 확인
4. **많은 단어 추가**: 10개 이상의 단어를 추가하고 모든 기능이 정상 작동하는지 확인
5. **통계 정확도**: 같은 단어로 여러 번 퀴즈를 풀고 통계가 정확히 기록되는지 확인

## 문제 해결

### 문제: "python을 찾을 수 없습니다"
**해결**: Python이 설치되어 있는지 확인하고 PATH에 추가되어 있는지 확인

### 문제: 한글이 깨져서 보입니다
**해결**: PowerShell의 인코딩 설정 변경
```powershell
chcp 65001
```

### 문제: 파일이 저장되지 않습니다
**해결**: 
- 프로그램이 실행되는 폴더에 쓰기 권한이 있는지 확인
- 파일 탐색기에서 해당 폴더를 확인
