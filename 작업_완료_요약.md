### 3단계: 단어 검색 기능 추가 (코드 실습) ✅

#### 구현 내용:
1. **HTML 추가**
   - 검색 입력창 추가
   - 검색 버튼 추가
   - 실시간 검색 지원 (onkeyup 이벤트)

2. **JavaScript 구현**
   - `searchWords()` 함수 추가
   - `displayWords()` 함수로 코드 재사용성 향상
   - 영어/한글 모두 검색 가능
   - 검색어 지우면 자동으로 전체 목록 표시

3. **CSS 스타일**
   - 검색 입력창 스타일
   - 검색 버튼 스타일
   - 다크 모드 지원

#### 코드 구조 이해:
- **프론트엔드**: 사용자 입력 → JavaScript 함수 → API 호출 → 화면 업데이트
- **백엔드**: API 요청 받기 → 데이터 처리 → JSON 응답 반환
- **데이터 흐름**: HTML → JavaScript → Flask API → JSON → JavaScript → HTML
