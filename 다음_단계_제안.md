## 🎯 추천 순서

### 1단계: 앱 실제 사용 및 피드백 수집 (우선 추천!)
**목적**: 실제로 사용해보면서 개선점 찾기

**할 일**:
1. 브라우저에서 앱 열기 (`http://127.0.0.1:5000`)
2. 실제로 단어 추가하고 퀴즈 풀어보기
3. 불편한 점이나 개선하고 싶은 점 기록
4. `기능_테스트_체크리스트.md` 파일로 체계적으로 테스트

**예상 소요 시간**: 30분~1시간

### 2단계: 실용적인 기능 추가

#### A. 다중 선택형 퀴즈 (추천!)
**왜 좋은가?**
- 객관식 문제로 더 재미있게 학습
- 코드 구조 이해에 도움 (랜덤 선택지 생성 로직)

**구현 내용**:
- 4지선다 문제 생성
- 정답과 오답 3개 랜덤 선택
- 선택지 섞기

#### B. 통계 시각화
**왜 좋은가?**
- 정답률을 그래프로 보기
- 학습 진도 한눈에 파악

**구현 내용**:
- Chart.js 라이브러리 사용
- 정답률 막대 그래프
- 단어별 통계 차트

#### C. 틀린 단어 집중 학습 모드
**왜 좋은가?**
- 약한 단어만 반복 학습
- 학습 효율 향상

**구현 내용**:
- 틀린 단어만 필터링
- 틀린 단어만 퀴즈로 출제
- 정답률 낮은 단어 우선 학습

#### D. 단어 카테고리/태그
**왜 좋은가?**
- 단어를 주제별로 분류
- 체계적인 학습 관리

**구현 내용**:
- 카테고리 추가/수정/삭제
- 단어에 카테고리 할당
- 카테고리별 필터링
