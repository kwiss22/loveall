# UI가 안 나올 때 해결 방법

## 🔍 문제 진단

예쁜 단어장 UI가 안 나오는 경우, CSS 파일이 제대로 로드되지 않았을 가능성이 높습니다.

---

## ✅ 단계별 해결 방법

### 1단계: 브라우저 하드 리프레시

1. **브라우저에서 Ctrl + Shift + R** (하드 리프레시)
   - 또는 **Ctrl + F5**
   - 또는 **F12** → 새로고침 버튼 오른쪽 클릭 → "캐시 비우기 및 강력 새로고침"

2. **주소 확인:**
   ```
   http://127.0.0.1:5000
   ```
   - `/main` 같은 경로가 붙어있지 않은지 확인

### 2단계: 브라우저 개발자 도구로 확인 (F12)

1. **F12 키** 눌러서 개발자 도구 열기

2. **Console 탭 확인:**
   - 빨간색 오류 메시지가 있는지 확인
   - 예: "Failed to load resource: style.css"
   - 오류 메시지를 기록해두세요

3. **Network 탭 확인:**
   - F5로 새로고침
   - `style.css` 파일 찾기
   - 상태 확인:
     - ✅ **200 OK** → 파일은 로드됨 (다른 문제)
     - ❌ **404 Not Found** → 파일 경로 문제
     - ❌ **Failed** → 서버 문제

### 3단계: CSS 파일 직접 확인

브라우저 주소창에 입력:
```
http://127.0.0.1:5000/static/style.css
```

**결과 확인:**
- ✅ CSS 코드가 보이면 정상 (파일 경로는 올바름)
- ❌ "Not Found" 오류가 나면 Flask의 static 폴더 설정 문제

### 4단계: Flask 앱 재시작

1. **터미널에서 Ctrl + C**로 Flask 앱 종료

2. **다시 실행:**
   ```powershell
   python web_vocab_app.py
   ```

3. **성공 메시지 확인:**
   ```
   * Running on http://127.0.0.1:5000
   ```

4. **브라우저에서 Ctrl + Shift + R** (하드 리프레시)

### 5단계: 브라우저 캐시 완전히 지우기

1. **Ctrl + Shift + Delete** 눌러서 캐시 삭제
2. **브라우저 닫기**
3. **새 브라우저 창 열기**
4. **주소 입력:** `http://127.0.0.1:5000`

### 6단계: 다른 브라우저로 시도

- Chrome, Edge, Firefox 등 다른 브라우저로 시도
- 같은 문제가 발생하는지 확인

## 🔧 추가 확인 사항

### 1. 올바른 Flask 앱이 실행 중인가?

터미널에서 다음 메시지가 보여야 합니다:
```
============================================================
영어 단어장 웹 애플리케이션 시작!
============================================================
* Running on http://127.0.0.1:5000
```

**중요:** `web_vocab_app.py`가 실행 중이어야 합니다!
- ❌ `test_flask_simple.py` (X)
- ✅ `web_vocab_app.py` (O)

### 2. 올바른 주소로 접속했나?

- ✅ `http://127.0.0.1:5000` (O)
- ❌ `http://127.0.0.1:5000/main` (X)

### 3. 파일 구조 확인

```
game_english/
├── web_vocab_app.py
├── templates/
│   └── index.html
└── static/
    ├── style.css  ← 이 파일이 있어야 합니다!
    └── script.js  ← 이 파일도 있어야 합니다!
```

## 💡 가장 빠른 해결 방법

### 방법 1: 완전히 새로 시작

1. **모든 터미널 창 확인**
   - 실행 중인 Flask 앱 모두 Ctrl + C로 종료

2. **Flask 앱 실행:**
   ```powershell
   python web_vocab_app.py
   ```

3. **브라우저에서:**
   - Ctrl + Shift + Delete (캐시 삭제)
   - 새 브라우저 창 열기
   - `http://127.0.0.1:5000` 접속
   - Ctrl + Shift + R (하드 리프레시)

### 방법 2: 시크릿 모드로 접속

1. **Ctrl + Shift + N** (Chrome) 또는 **Ctrl + Shift + P** (Edge)
2. **주소 입력:** `http://127.0.0.1:5000`
3. **Enter 키**

---

## ✅ 정상 작동 확인

CSS가 제대로 로드되면:
- ✅ **보라색 그라데이션 배경**이 보여야 합니다
- ✅ **흰색 카드 형태의 컨테이너**가 보여야 합니다
- ✅ **보라색 헤더**에 "영어 단어장" 제목이 보여야 합니다
- ✅ **탭 버튼** (단어 목록, 단어 추가, 퀴즈, 통계)이 보여야 합니다
- ✅ **예쁘게 디자인된 화면**이 보여야 합니다

## 🔍 일반적인 원인과 해결

### 원인 1: 브라우저 캐시
**해결:** Ctrl + Shift + R (하드 리프레시)

### 원인 2: CSS 파일 경로 문제
**해결:** `http://127.0.0.1:5000/static/style.css` 직접 접속 확인

### 원인 3: 잘못된 Flask 앱 실행
**해결:** `python web_vocab_app.py` 실행 확인

### 원인 4: 파일이 없음
**해결:** `static/style.css` 파일 존재 확인
